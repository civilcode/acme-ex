# Naming Conventions

## Dates and Timestamps

| type     | suffix | example      |
| -------- | ------ | ------------ |
| Date     | _on    | completed_on |
| DateTime | _at    | competed_at  |

* Never use `date` or `datetime` as field names.

## Paths

| term          | example            |
| ------------- | ------------------ |
| root_path     | `/`                |
| relative_path | `./bar/baa.txt`    |
| absolute_path | `/foo/bar/baa.txt` |
| parent_path   | `/foo/bar`         |
| filename      | `baa.txt`          |
| basename      | `baa`              |
| extension     | `.txt`             |
