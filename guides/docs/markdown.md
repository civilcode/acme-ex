# Markdown Guide

https://www.markdownguide.org/basic-syntax

There are a number of alternatives for using markdown, e.g. when defining a header there are
two options:


    # Heading level 1

    Heading level 1
    ===============

This guide specifies our preferences:

## Headings

    # Head level 1

## Bullets

    - Bullet 1

(easy to convert to task lists)

## Bold

    **bold**

## Italics

    _italics_

## Ordered Lists

    1. Item 1
    2. Item 2
